# Supermarket Management System

A supermarket management system implemented using data structures in C++.

## Features
- Product management
- Shopping cart system
- Binary Search Tree for product storage
- Hash Table for fast product lookup
- Customer queue simulation

## Technologies Used
- C++
- Data Structures (BST, Hash Table, Stack, Queue, Linked List)

## How to Run
1. Compile the C++ file using any compiler (g++ or Visual Studio)
2. Run the generated executable
3. Use the menu to interact with the system

## About
This project demonstrates the use of data structures in building real-world applications.
